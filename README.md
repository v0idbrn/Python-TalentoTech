## 🎯 Course Objectives
The program focuses on mastering Python's logic and syntax to solve complex problems and create scalable solutions. The curriculum is designed to move from basic algorithmic thinking to full database integration.

## 🛠️ Technical Stack
* **Language:** Python 3.12+
* **Database:** SQLite3 (SQL)
* **Environment:** Visual Studio Code (VS Code)
* **Version Control:** Git & GitHub

## 📋 Syllabus & Learning Modules
The course is structured into 16 intensive classes covering:
- **Phase 1: Fundamentals**: Variables, operators, and basic I/O operations.
- **Phase 2: Flow Control**: Logical reasoning using `if/else/match` and iteration loops (`while/for`).
- **Phase 3: Data Structures**: Advanced manipulation of Strings, Lists, Tuples, and Dictionaries.
- **Phase 4: Modularity**: Designing reusable code with Functions, Scope management, and External Modules.
- **Phase 5: Persistence & Databases**: File handling (.txt) and full integration with **SQLite** for relational data management.

## 🏁 Capstone Project: Inventory Management System
The final deliverable is a functional **Inventory Management System** designed for terminal execution, featuring:
* **Relational Database**: Persistent storage using SQLite to manage products, categories, and pricing.
* **Full CRUD Logic**: Implementation of Create, Read, Update, and Delete operations.
* **Business Logic**: Automated reporting system for low-stock alerts and inventory valuation.
